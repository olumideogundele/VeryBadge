Badge Plugin Demo

The Badge plugin allows you to inform the user of your app that you have something new for them while the application isn't running in the foreground. How this is indicated to the user depends on the device platform and version.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/Windows Phone - App package.
